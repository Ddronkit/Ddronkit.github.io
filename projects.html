LinkVan - Sept VanHacks 2019

I've joined the VanHacks 2019 to make an app for the non-profit linkVan.
Linkvan is a mobile application designed for people with low income who have limited access to wifi. The app provides user with locations of basic servies inluding food, shelters, and wifi. It also gives user information about events host by other non-profit dedicated to help with their situation, such as job oppoirtunities if they are looking for employment.

This app is built with React Native with Prisma and GraphQL for backend

Mr.Ping & Mr.Pong - June 2019

I got interested in how old games are made, so I followed through a tutorial on udemy and built the basic of this arcade game.
I then added keyboard control on top of the mouse control for 2 people to play together, though the left player would be very annoyed if the right player choose to fiddle aroudn with the mouse in the middle of the match.
I've also added a menu screen for aesthetics and the completion of the game. Player can change from single player mode to two-player mode and adjust ball speed or AI tracking speed.

This game is built in Javascript and you can try it yourself here.
