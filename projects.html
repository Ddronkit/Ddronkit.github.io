LinkVan - Sept VanHacks 2019

I've joined the VanHacks 2019 to make an app for the non-profit linkVan.
Linkvan is a mobile application designed for people with low income who have limited access to wifi. The app provides user with locations of basic servies inluding food, shelters, and wifi. It also gives user information about events host by other non-profit dedicated to help with their situation, such as job oppoirtunities if they are looking for employment.

This app is built with React Native with Prisma and GraphQL for backend
